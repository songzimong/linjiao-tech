# 玄武蜂群安全护栏 (Xuanwu Swarm Guard)

> 群星体系 v1.0 | 2026-06-22

## 三层防护

### 第一层：事前预防
- 蜂群启动前，玄武审查议题是否在安全范围内
- 划定 Agent 的对话边界（可以讨论什么、不能讨论什么）
- 设置关键词黑名单

### 第二层：事中监控
- 玄武作为**旁路监听者**，不参与对话但实时读取所有消息
- 触发条件：
  - 出现敏感关键词 → 暂停蜂群
  - 对话偏离议题 → 标记警告
  - 某 Agent 行为异常 → 隔离该 Agent
  - 连续 3 次警告 → 终止蜂群

### 第三层：事后审计
- 蜂群结束后，玄武审查所有对话记录和输出
- 标记任何可疑内容
- 生成安全报告

## 安全等级

| 等级 | 含义 | 措施 |
|------|------|------|
| GREEN | 正常 | 无需干预 |
| YELLOW | 警告 | 标记对话，通知麒麟 |
| RED | 危险 | 立即终止蜂群，人工审查 |

## 关键词规则

玄武维护一份动态关键词库：
- 硬黑名单（触发 RED）：赌博、政治敏感、违法内容
- 软黑名单（触发 YELLOW）：争议性话题、未经验证的信息
- 白名单豁免：经麒麟授权的特定议题可豁免

## 异常检测

监测以下模式：
1. **循环对话**：相同观点反复出现超过 5 次
2. **群体极化**：所有 Agent 倾向同一极端观点
3. **沉默爆发**：某 Agent 突然停止发言（可能被劫持）
4. **信息污染**：传播未经证实的数据
