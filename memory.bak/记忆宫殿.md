## 共享记忆总线 (Shared Memory Bus)
> Hermes 融合特性：所有 Agent 可读写此总线

### 活跃上下文
- 当前项目状态、正在进行中的任务
- 由麒麟在每次任务分发时注入

### 记忆快照
- 每个 Agent 的最新状态摘要存储在 `bus/agent-status.json`
- 关键决策记录在 `bus/decisions.json`

## 全局检索
- 长期记忆：`../MEMORY.md`
- 每日记录：`memory/YYYY-MM-DD.md`
- 各殿索引提供领域内快速检索
- 共享总线提供实时 Agent 状态
